About
We Play Games is a website that allows the user to play many different types of video games. Currently we are running the old N64 Pokeman Stadium game. This can be played with two players across mutliple users on the internet.
How to Play

Input Commands

Input Button Pressed
a A Button
b B Button
start Start Button
r Right Trigger
l Left Trigger

Analog Stick Commands

Input Analog Controls
up Directional Up
down Directional Down
left Directional Left
right Directional Right

C-Button Commands

Input C Buttons Pressed
cup Up Button
cdown Down Button
cleft Left Button
cright Right Button
